# ⚽ FIFA Player Value Analysis

## Project Overview

This project explores the relationship between FIFA player ratings and market value using SQL, Excel, and Tableau. After cleaning and organizing a dataset of over 10,000 player records, I created an interactive dashboard to uncover trends and support data-driven decision-making.

---

## Business Problem

Football clubs invest heavily in player transfers. This project analyzes which player attributes are most strongly associated with market value and demonstrates how data can support scouting and transfer decisions.

---

## Key Findings

- Overall player rating is a strong predictor of market value.
- Data cleaning improved dataset consistency and analysis quality.
- Interactive dashboards make it easier to identify trends across leagues, positions, and player performance.
